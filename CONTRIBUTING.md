# Contributing to Will You Be My Valentine 2026

Thanks for your interest in making this project even cuter!

## How to Contribute

1. **Fork** this repository
2. **Clone** your fork locally
3. **Create a branch:**
   ```bash
   git checkout -b feature/your-idea
   ```
4. **Make your changes** and test in a browser
5. **Commit** with a clear message
6. **Push** and **open a Pull Request**

## Ideas for Contributions

- New animation effects
- Additional themes (dark mode, other aesthetics)
- Accessibility improvements
- New celebration page variations
- Internationalization (multi-language support)
- Performance optimizations

## Guidelines

- Keep it vanilla — no frameworks or build tools
- Test on mobile and desktop browsers
- Keep the coquette/lovecore aesthetic consistent
- Don't break offline support (Service Worker)
